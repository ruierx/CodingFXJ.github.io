# Github 配置

  • 创建一个仓库，名为：自己用户名 + .github.io

  • 配置 SSH key，复制生成的密钥（在 C:\Users\admin\.ssh\id_rsa.pub 文件里），

    # Github—>Setting—>SSH and GPG keys —>new SSH keys，粘贴并添加 SSH keys。

# 部署遇到的问题及解决方式

hexo d 遇到问题一：删除博客根目录下的.deploy_git 文件，重新部署（ hexo cl && hexo g && hexo d）

hexo d 遇到问题二： OpenSSL SSL_read: Connection was reset, errno 10054

解决方法： git config --global http.sslVerify "false"
