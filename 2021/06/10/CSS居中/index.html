# CSS 细节

# 1. 浮动的细节（扩展）

  • 浮动盒子的顶边不得高于上一个盒子的顶边

  • 若剩余空间无法放下浮动的盒子，则该盒子向下移动，直到具备足够的空间能容纳盒子，然后再向左或向右移动

# 2. 行高的取值

  • 数字的时候，子元素的行高 = 子元素的字体大小 * 数字

    # (行高是先继承下来，再乘以当前元素的字体大小来计算当前行高)

  • 百分比和 em 单位时，子元素的行高 = 父元素的字体大小 * 百分比（或者 em 单位的值）

    # (行高是先乘以祖先元素的字体大小计算好结果，再继承下来作用在子元素上)

# 3. 多个行盒垂直方向上的对齐

​ 给没有对齐元素设置 vertical-align：预设值/px

# 4. 图片的底部白边

解决方案：

1. 设置父元素的字体大小为0

2. 将图片设置为块盒

# 5. 堆叠上下文

堆叠上下文（stack context），它是一块区域，这块区域由某个元素创建，它规定了该区域中的内容在 Z 轴上排列的先后顺序。

创建堆叠上下文的元素

  1. html 元素（根元素）

  2. 设置了 z-index（非 auto）数值的定位元素

同一个堆叠上下文中元素在 Z 轴上的排列

从后到前的排列顺序：

  1. 创建堆叠上下文的元素的背景和边框

  2. 堆叠级别 (z-index, stack level) 为负值的堆叠上下文

  3. 常规流非定位的块盒

  4. 非定位的浮动盒子

  5. 常规流非定位行盒

  6. 任何 z-index 是 auto 的定位子元素，以及 z-index 是 0 的堆叠上下文

  7. 堆叠级别为正值的堆叠上下文

# 6. 浏览器兼容性

# 水平居中

​ 居中：盒子在其包含块中居中

# 1. 行盒（行块盒）水平居中

​ 直接设置行盒（文本 / 行块盒）父元素 text-align:center

# 2. 常规流块盒水平居中

  • 定宽，设置左右 margin 为 auto

优点：只需要对子元素进行设置，就可以现实水平方向居中布局效果
缺点：如果子级脱离文档流，会导致 margin 属性的值无效 。

父元素设置：display: table，子元素设置 margin: 0 auto; 也可以实现

  • 不定宽，display + text-align

父元素设置 text-align: center ，子元素设置 display：inline-block

提示: text-algin 具有继承性

# 3. 块元素的水平居中

  • 定宽 ：定位 + margin
  • 定宽：子元素的 margin-left: - width / 2 px ;
  • 不定宽：利用 css3 新增属性 transform: translateX(-50%);

absolute + transform 方案的优缺点：
优点：不论父级元素是否脱离文档流，都不会影响子级元素的的水平居中效果。
缺点：transform 属性时 CSS3 新增的属性，老版本浏览器兼容情况不太好。

  • 使用 flexbox 布局实现（宽度定不定都可以）

​ 给待处理的块状元素的父元素添加属性 display: flex; justify-content: center;

# 垂直居中

# 1. 单行文本的垂直居中

​ 设置文本所在元素的行高 ( line-height )，为整个区域的高度

# 2. 多行文本的垂直居中 (行块盒、块盒内)

​ (未知高度）多行文本垂直居中设置上下的 padding 值一样即可。

  • 多行文本固定高度的居中（模拟 table 方法）

    父元素使用 display: table 和子元素使用 display: table-cell

  • 对子元素设置 display: inline-block 属性，父元素设置对应的 height 和 line-height。

    对子元素设置 vertical-align: middle 属性，使其基线对齐。

  • 绝对定位 + transform

给文本套一个 p 元素

注意：文本超过父元素不管用

# 3. 块盒绝对定位的垂直居中

  • 定高，设置上下的坐标为 0（ top:0, bottom:0 ），将上下 margin 设置为 auto
  • absolute + 负 margin 方式
  • absolute + transform(0, - 50%)

# 水平垂直居中

# 一.、已知子元素高度和宽度的元素

  1. 设置父元素为相对定位，给子元素设置绝对定位，

top: 0; right: 0; bottom: 0; left: 0;

margin: auto;

  1. 设置父元素为相对定位，给子元素设置绝对定位

left: 50% ; top: 50% ;

margin-left: - 元素宽度的一半px ;

margin-top: - 元素高度的一半px

# 二、 未知子元素高度和宽度的元素

  1. 设置父元素为相对定位，给子元素设置绝对定位，

left: 50%;

top: 50%;

transform: translateX(-50%) translateY(-50%);

  1. 使用 flex 布局实现
  1. 利用 table-cell;
  1. 利用 grid 布局
  1. flex+margin
  1. JS 模拟 CSS 实现
