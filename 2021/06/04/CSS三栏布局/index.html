CSS三栏布局

|字数总计:856|阅读时长:4分钟|阅读量:

# 三栏布局

共同目的：实现两侧宽度固定，中间宽度自适应的三栏布局，并且中间先加载渲染。

# 一、双飞翼布局

主要思路： 浮动+负margin

给主区域添加一个父元素，设置 width：100% ，

让左中右三个区域都左浮动，

给主区域设置左右 margin 缩进，留出空位给两边的区域，

让左边区域 margin-left:-100% ，右边区域 margin-left:-200px

# 二、圣杯布局

# 方式一

主要思路： 浮动+负margin

用一个父元素包裹三个区域，让三个元素左浮动，

父元素清除浮动，并设置左右内边距空出两边区域的宽度，

左区域： margin-left:-100% ；右区域： margin-left：-200px

# 方式二

主要思路： 浮动+相对定位

用一个父元素三个区域，让三个元素左浮动，

父元素清除浮动，并设置左右内边距空出两边区域的宽度，

给左右区域添加相对定位，都相对自身位置，偏移自己宽度位移

# 三、flex 布局

主要思路：

让父元素包裹左中右三个元素，并设置父元素为 flex 容器，

通过项目的属性 order 属性来设置 order: -1 ，让左区域排在中间区域前面

项目属性 flex-grow:1 设置 main 的放大比例，填充剩余空间，默认为 0

通过项目属性 flex-basis 设置左右的固定宽度

# 四、绝对定位

主要思路：

给父元素添加相对定位，

中间设置 margin 留出两边空位

左右给绝对定位进行摆放

中间设置左右 margin 留出空位
