# 属性

属性：对象身上保存的变量就是属性
操作属性：
对象。属性名称 = 值；
对象 [“属性名称”] = 值；

# 属性节点

属性节点：DOM 元素的 attributes 属性中保存的所有内容都是属性节点

操作属性节点：

DOM 元素.setAttribute (“属性名称”, “值”);
DOM 元素.getAttribute (“属性名称”);

# 属性和属性节点区别：

任何对象都有属性，但是只有 DOM 对象才有属性节点

# jQuery 属性和属性节点操作

# 属性节点获取与赋值

attr(name | pro | key,val | fn)、prop()
作用：获取或者设置属性节点的值
可以传递一个或多个参数
传递一个参数，获取属性节点的值
传递两个参数，设置属性节点的值

注意点:

如果是获取：无论找到多少个元素，都只会返回第一个元素指定的属性节点的值
如果是设置：找到多少个元素就会设置多少个元素
如果是设置：设置的属性节点不存在，那么系统会自动新增

# 删除属性节点

removeAttr(name)、removeProp()

prop () 与 attr ()

prop 方法不仅能够操作属性，他还能操作属性节点

在操作属性节点时，具有 true 和 false 两个属性的属性节点，如 checked, selected 或者 disabled 推荐使用 prop ()，

其他的使用 attr ()

# 操作类相关方法

addClass (class|fn) 添加一个类，添加多个，多个类名之间用空格隔开

removeClass ([class|fn]) 删除一个类，除多个，多个类名之间用空格隔开

toggleClass (class|fn [,sw]) 有就删除，没有就添加

# 动画效果

自定义动画 animate ()

animate( params, [speed], [easing], [fn])

params: 一组包含作为动画属性和终值的样式属性和及其值的集合，多个属性的动画也会同时执行
speed: 指定动画时长 (预定值：“slow”,“normal”, or “fast”) 或毫秒数
easing: 指定动画节奏，默认就是 swing （linear：匀速）
fn: 动画执行完毕之后的回调函数，每个元素执行一次。

delap 方法：延迟动画执行

stop 方法：停止动画执行
