# Vue 常用属性

# 整体总结

# 计算属性 computed

计算属性是基于它们的响应式依赖进行缓存的，只在相关响应式依赖发生改变时它们才会重新求值

当其他的依赖数据没有发生改变，它调用的是缓存的数据，极大的提高了程序的性能

# computed 与 methods 的区别

computed:

  1. 只对 data 数据进行复杂逻辑处理，

  2. 初始化执行过后，只要任何值有更新，computed 和其相关的值都会更新，

  3. 执行会被缓存起来，不必在数据没有发生改变的时候重新获取数据的值，效率高

  4. 不能执行异步操作

methods :

调用的时候才会执行对应的方法，每次都会重新计算，不会自动同步数据

# 计算属性的 setter/getter

计算属性默认只有 getter，不过在需要时你也可以提供一个 setter

# 监听属性 watch

watch 中可以执行任何逻辑，如函数节流，Ajax 异步获取数据，甚至操作 DOM（不建议）

三个参数：

handler 回调函数

deep ：Boolean 值

immediate ：Boolean 值

immediate 为 true 时，当组件初始化时就已经开始监听（即 beforeCreate 阶段）

​ 为 false 时，只有改变时才会监听

deep 可以检测到对象内部属性的变化。

# watch 与 computed

watch：监测的是属性值， 只要属性值发生变化，其都会触发执行回调函数来执行一系列操作。

computed：监测的是依赖值，依赖值不变的情况下其会直接读取缓存进行复用，变化的情况下才会重新计算。

注意：计算属性不能执行异步任务，计算属性必须同步执行。也就是说计算属性不能向服务器请求或者执行异步任务。

# 过滤器 filters

用于格式化变量内容的输出。（日期格式化，字母大小写，数字再计算等等）
