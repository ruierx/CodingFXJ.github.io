# 1 前言

CSS3 相比 CSS 2 对边框、背景、渐变、文本效果、字体、转换效果、过渡、动画、图片、用户界面、盒模型、媒体查询等都有了很大的改进，新增了不少功能和属性

# 4 CSS3 渐变

# 4.1 语法

linear-gradient

background: linear-gradient(direction, color-stop1, color-stop2, …);

background: linear-gradient( red, blue);

direction:

从左到右: to right

从左上角开始到右下角: to bottom right

指定角度：角度是指水平线和渐变线之间的角度（0deg：从下到上 /90deg 从左到右)

多个颜色： background: linear-gradient (to right, red,orange,yellow,green,blue,indigo,violet);

使用透明度：background: linear-gradient (to right, rgba (255,0,0,0), rgba (255,0,0,1));

repeating-linear-gradient () 函数用于重复线性渐变

# 4.2 CSS3 径向渐变

不同尺寸大小关键字的径向渐变

重复的径向渐变，repeating-radial-gradient () 函数用于重复径向渐变

background: repeating-radial-gradient(red, yellow 10%, green 15%);

# 5 CSS3 文本效果

  • # text-shadow：适用于文本阴影
  • # box-shadow：适用于盒子阴影
  • # text-overflow：指定应向用户如何显示溢出内容
  • word-wrap：自动换行属性允许您强制文本换行
  • word-break 指定单词拆分换行规则

normal：使用浏览器默认的换行规则

break-all：允许在单词内换行

keep-all：只能在半角空格或连字符处换行

# 6 CSS3 字体

CSS3 中用 @font-face 属性来设置自己喜欢的字体

在新的 @font-face 规则中，您必须首先定义字体的名称（比如 myFirstFont），然后指向该字体文件

# 7 CSS3 2D 转换

# 7.1 属性

  • transform
  • transform-origin

# 7.2 方法

  • # translate ()：根据左 (X 轴) 和顶部 (Y 轴) 位置给定的参数，从当前元素位置移动
  • # rotate ()：给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转
  • scale()：元素增加或减少的大小，取决于宽度（X 轴）和高度（Y 轴）的参数
  • # skew()：
  • skewX (); 表示只在 X 轴 (水平方向) 倾斜。
  • skewY (); 表示只在 Y 轴 (垂直方向) 倾斜。
  • transform: skew (相对于 x 轴角度，相对于 y 轴角度)
  • transform: skew(30deg,20deg);
  • # matrix ()：将 2D 变换方法合并成一个

# 8 CSS3 3D 转换

# 8.1 转换属性

  • transform：向元素应用 2D 或 3D 转换。

  • perspective：规定 3D 元素的透视效果。

# 8.2 3D 转换方法

  • translate3d(x,y,z)

  • scale3d(x,y,z)

  • perspective(n)

# 9 CSS3 过渡

# 9.1 属性

transition：简写属性， 设置四个过渡属性

transition-property：规定应用过渡的 CSS 属性的名称

transition-duration：定义过渡效果花费的时间。默认是 0

transition-timing-function：规定过渡效果的时间曲线。默认是 “ease”

transition-delay：规定过渡效果何时开始。默认是 0。

# 10 CSS3 动画

# 10.1 属性

  • @keyframes
  • animation

# 10.2 语法

**animation：** 动画名称，一个周期花费时间，运动曲线（默认 ease），动画延迟（默认 0），播放次数（默认 1），是否反向播放动画（默认 normal），是否暂停动画（默认 running）

示例 1：

/ 执行一次 logo2-line 动画，运动时间 2 秒，运动曲线为 linear/
animation: logo2-line 2s linear;

示例 2：

/2 秒后开始执行一次 logo2-line 动画，运动时间 2 秒，运动曲线为 linear/
animation: logo2-line 2s linear 2s;

# 重要属性

animation-fill-mode : none | forwards | backwards | both;
/*none：不改变默认行为。
forwards ：当动画完成后，保持最后一个属性值（在最后一个关键帧中定义）。
backwards：在 animation-delay 所指定的一段时间内，在动画显示之前，应用开始属性值（在第一个关键帧中定义）。
both：向前和向后填充模式都被应用。 */

# 11 CSS3 用户界面

  • **resize：**resize 属性指定一个元素是否是由用户调整大小的。

resize: none|both|horizontal|vertical

none：用户无法调整元素的尺寸。

both：用户可调整元素的高度和宽度。

horizontal：用户可调整元素的宽度。

vertical：用户可调整元素的高度

  • # box-sizing
  • # outline-offset

在边框 border 外面又加了一层 outline-offset

轮廓与边框有两点不同：

  • 轮廓不占用空间
  • 轮廓可能是非矩形
