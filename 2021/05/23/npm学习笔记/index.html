# npm 介绍

npm（Node Package Manager）是 node 的包管理器，它是随着 NodeJs 安装时一起被安装。

# npm 作用

通过 NPM 可以对 Node 的工具包进行搜索、下载、安装、删除、上传。借助别人写好的包，方便于开发。
