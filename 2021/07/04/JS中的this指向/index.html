函数内this指向

调用方式 this指向
普通函数调用 window
构造函数调用 实例对象
对象方法调用 该方法所属的实例对象
事件绑定方法 绑定事件对象
定时器函数 window
立即执行函数 window
箭头函数 函数定义位置的上下文this
严格模式下全局作用域中函数中的 this undefined

改变this指向

常用的 三种方法：bind()、call()、apply()

传参形式 参数和返回值 执行时机 使用场景
fun.call(thisArg, arg1, arg2, …) 函数的返回值，即是调用函数 立即执行 通常用于继承
fun.apply(thisArg, [argsArray]) 必须是数组，函数的返回值 立即执行 数组处理
fun.bind(thisArg, arg1, arg2, …) 返回指定this和参数的新函数 不会立即执行函数 改变定时器内部的this指向
