# 原生 Dom 对象和 jQuery 对象区别

​ 联系 — 两者之间可互相转换

1、jQuery 对象可以通过 jQuery 包装 DOM 对象后产生；

2、DOM 对象也可以通过 jQuery 按索引取得；

​ 区别 — 两个对象完全不同

1、jQuery 对象和原生 JS 中的 dom 对象两者对象类型，js 原生获取的 dom 是一个对象，jQuery 对象就是一个伪数组，两者不等价；

2、jQuery 无法使用 DOM 对象的任何方法，同理 DOM 对象也不能使用 jQuery 里的方法。
